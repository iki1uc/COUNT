# COUNT

COUNT ist ein UI‑System, das den RAUM‑ID‑Kernel nutzt, um
RESPO‑Cluster zu scannen und Systemlagen darzustellen.

## Dateien

- index.html – Hauptscanner
- UI-SCANNER.html – UI‑Oberfläche
- ID.html – ID‑Matrix
- raum.html – RAUM‑Modul
- horizontal.html – horizontale Achse
- waagarecht.html – waagerechte Achse
- fin.e.js – Kernel (GHOST_SCAN_REAL, RAUM_KERNEL, MOVE)
- RESPO/ – Cluster‑Daten

## Funktion

COUNT lädt RESPO‑Cluster und zeigt:

- ID
- LAGE
- STATUS
- RESULT
- OUT
- WAHRHEIT
- EDIT

COUNT ist vollständig kompatibel mit RAUM‑ID.
